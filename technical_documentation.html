Introduction

HTML is an acronym that is short for "Hyper Text Markup Language". Basically, this is the written language that your internet browser translates into the many different things you see on the internet. It is all based on HTML, including this page you are reading now! The great thing about learning this language is that it uses your current language and the commands are super logical.

In this document I am going to go over the basics that you will need to get started. Please keep in mind this is not a "teaching" document so much as it is a guidline with very little information. That being said, a lot of the things mentioned here will help you know what to search for to get more information.

HTML Elements

An HTML Element is a piece or section of your complete page. You can tell an element as it is usually surrounded by a less than and a greater than sign like this- 

      
        <header>
Most Elements need to have an opening "Tag" and a closing "Tag" This dictates where the element begins and ends. You close an element almost the same way you open it, with one key difference- 
        
          </header>
All of the text that you put between the two "Tags" is what will show up on the viewers screen!

Different "Tags" tell the browser what type of content is in between them. Here are some examples.

  • <header> Tells the browser that this is a "Headline".
  • <p> Tells the browser that this is a "Paragraph".
  • <main> Is what the browser uses to identify what your page is about. Most of your other "Tags" and content will be "nested" within your "Main" tags.

Links

Being able to link to another page or even a different section of your page is important to know as well!

You can link and image from another page, or you can link a video. You can also link to a source of information such as Wikipedia.

  • To link an image you use the "img" and "src" tags together. It looks like this 
          
      <img src="https//www.yourimagesource.com/img.jpg">
  • You can also create a link that takes a user to a specific section of your page. To do this you use the "a" tag, which stands for anchor. This tag is always used with the "href" attribute, which tells the anchor where it is going. 
        
    <a href="#introduction">Introduction
    • The text between the > and the closing "anchor" tag will provide the link that needs to be clicked on. Just remember, internal links to your page use "href" and links to external sources use the "src" tag.
